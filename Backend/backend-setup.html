5 Set Up PostgreSQL (Docker Compose)

We use a docker-compose.yml at the project root instead of a raw docker run command. This is better because:

  • You don't have to remember long commands — just docker compose up -d
  • A named volume (pgdata) keeps your data safe across container restarts
  • It's committed to the repo, so anyone can spin up the same environment
  • Easy to add more services later (e.g. Redis, pgAdmin)

The docker-compose.yml lives at the project root (new-project/docker-compose.yml):

services:
  db:
    image: postgres:16
    container_name: cybersec-db
    restart: unless-stopped
    environment:
      POSTGRES_USER: admin
      POSTGRES_PASSWORD: password
      POSTGRES_DB: cybersec_sip
    ports:
      - "5432:5432"
    volumes:
      - pgdata:/var/lib/postgresql/data

volumes:
  pgdata:

Start the Database

From the project root (new-project/), run:

docker compose up -d

Verify it's running:

docker compose ps

You should see cybersec-db with status "running". If the status shows an error, check logs with docker compose logs db.

Common Commands

CommandWhat it does
docker compose up -dStart the database in the background
docker compose downStop the database (data is preserved in volume)
docker compose down -vStop AND delete all data (nuclear option)
docker compose logs dbView database logs
No Dockerfile needed. We're using the stock postgres:16 image as-is — a Dockerfile is only needed when building a custom image.

9 Create the FastAPI App

This is where you create the actual API server. You need to create the following files inside Backend/:

Backend/app/
├── __init__.py    # Makes app/ a Python package (required for imports)
└── main.py        # FastAPI entry point

main.py is the entry point that:

  • Connects to your database via Prisma on startup (and disconnects on shutdown)
  • Sets up CORS so your Next.js frontend (localhost:3000) can call the API
  • Exposes a /health endpoint to verify the server is running

app/__init__.py

This file can be empty. It tells Python that app/ is a package, which is required for uvicorn app.main:app to resolve the import path.

# This file is intentionally left empty.

app/main.py

from contextlib import asynccontextmanager
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
from prisma import Prisma

db = Prisma()

@asynccontextmanager
async def lifespan(app: FastAPI):
    await db.connect()
    yield
    await db.disconnect()

app = FastAPI(
    title="Cybersecurity Sales Intelligence API",
    version="0.1.0",
    lifespan=lifespan,
)

app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:3000"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

@app.get("/health")
async def health():
    return {"status": "ok"}

11.1 Pydantic Schemas — app/schemas/company.py

Pydantic models define what data the API accepts and returns. There are three models per entity:

  • CompanyCreate — fields required when creating a new company
  • CompanyUpdate — all fields optional (partial update via PATCH)
  • CompanyResponse — the shape returned to the frontend
from datetime import datetime
from enum import Enum
from typing import Optional

from pydantic import BaseModel, Field


class CompanyStatus(str, Enum):
    prospect = "prospect"
    active_client = "active_client"
    previous_client = "previous_client"
    lost = "lost"
    disqualified = "disqualified"


class CompanySize(str, Enum):
    SMB = "SMB"
    Mid_Market = "Mid_Market"
    Enterprise = "Enterprise"
    Government = "Government"


class CompanyCreate(BaseModel):
    current_name: str = Field(..., min_length=1, max_length=255)
    status: CompanyStatus = CompanyStatus.prospect
    company_size: Optional[CompanySize] = None
    employee_count: Optional[int] = Field(None, ge=0)
    revenue_range: Optional[str] = None
    website: Optional[str] = None
    stock_ticker: Optional[str] = None
    country: Optional[str] = None


class CompanyUpdate(BaseModel):
    current_name: Optional[str] = Field(None, min_length=1, max_length=255)
    status: Optional[CompanyStatus] = None
    company_size: Optional[CompanySize] = None
    employee_count: Optional[int] = Field(None, ge=0)
    revenue_range: Optional[str] = None
    website: Optional[str] = None
    stock_ticker: Optional[str] = None
    country: Optional[str] = None


class CompanyResponse(BaseModel):
    id: str
    current_name: str
    status: CompanyStatus
    company_size: Optional[CompanySize] = None
    employee_count: Optional[int] = None
    revenue_range: Optional[str] = None
    website: Optional[str] = None
    stock_ticker: Optional[str] = None
    country: Optional[str] = None
    created_at: datetime
    updated_at: datetime

    class Config:
        from_attributes = True

11.2 Companies Router — app/routers/companies.py

The router provides five CRUD endpoints:

MethodEndpointDescription
POST/api/companiesCreate a new company
GET/api/companiesList companies (with filter & pagination)
GET/api/companies/{id}Get a single company by ID
PATCH/api/companies/{id}Update a company (partial)
DELETE/api/companies/{id}Delete a company
 
from typing import Optional

from fastapi import APIRouter, HTTPException, Query

from app.main import db
from app.schemas.company import (
    CompanyCreate,
    CompanyResponse,
    CompanyStatus,
    CompanyUpdate,
)

router = APIRouter(prefix="/api/companies", tags=["Companies"])


@router.post("/", response_model=CompanyResponse, status_code=201)
async def create_company(body: CompanyCreate):
    company = await db.company.create(
        data={
            "currentName": body.current_name,
            "status": body.status.value,
            "companySize": body.company_size.value if body.company_size else None,
            "employeeCount": body.employee_count,
            "revenueRange": body.revenue_range,
            "website": body.website,
            "stockTicker": body.stock_ticker,
            "country": body.country,
        }
    )
    return _to_response(company)


@router.get("/", response_model=list[CompanyResponse])
async def list_companies(
    status: Optional[CompanyStatus] = Query(None),
    skip: int = Query(0, ge=0),
    take: int = Query(20, ge=1, le=100),
):
    where = {}
    if status:
        where["status"] = status.value

    companies = await db.company.find_many(
        where=where, skip=skip, take=take,
        order={"createdAt": "desc"},
    )
    return [_to_response(c) for c in companies]


@router.get("/{company_id}", response_model=CompanyResponse)
async def get_company(company_id: str):
    company = await db.company.find_unique(where={"id": company_id})
    if not company:
        raise HTTPException(status_code=404, detail="Company not found")
    return _to_response(company)


@router.patch("/{company_id}", response_model=CompanyResponse)
async def update_company(company_id: str, body: CompanyUpdate):
    existing = await db.company.find_unique(where={"id": company_id})
    if not existing:
        raise HTTPException(status_code=404, detail="Company not found")

    update_data = {}
    if body.current_name is not None:
        update_data["currentName"] = body.current_name
    if body.status is not None:
        update_data["status"] = body.status.value
    if body.company_size is not None:
        update_data["companySize"] = body.company_size.value
    if body.employee_count is not None:
        update_data["employeeCount"] = body.employee_count
    if body.revenue_range is not None:
        update_data["revenueRange"] = body.revenue_range
    if body.website is not None:
        update_data["website"] = body.website
    if body.stock_ticker is not None:
        update_data["stockTicker"] = body.stock_ticker
    if body.country is not None:
        update_data["country"] = body.country

    if not update_data:
        return _to_response(existing)

    company = await db.company.update(
        where={"id": company_id}, data=update_data,
    )
    return _to_response(company)


@router.delete("/{company_id}", status_code=204)
async def delete_company(company_id: str):
    existing = await db.company.find_unique(where={"id": company_id})
    if not existing:
        raise HTTPException(status_code=404, detail="Company not found")
    await db.company.delete(where={"id": company_id})
    return None


def _to_response(company) -> dict:
    """Maps Prisma camelCase fields to snake_case API response."""
    return {
        "id": company.id,
        "current_name": company.currentName,
        "status": company.status,
        "company_size": company.companySize,
        "employee_count": company.employeeCount,
        "revenue_range": company.revenueRange,
        "website": company.website,
        "stock_ticker": company.stockTicker,
        "country": company.country,
        "created_at": company.createdAt,
        "updated_at": company.updatedAt,
    }

Important Note: Prisma + Python

Heads up: The prisma-client-py package is a community-maintained Python wrapper around Prisma. It works well but is not as mature as Prisma's official Node.js client. If you run into issues, the main alternative is SQLAlchemy (the standard Python ORM). However, Prisma gives you a single schema definition that can be shared conceptually with your Next.js frontend.
